Python 批量重命名文件

发表于 2021-06-07 分类于 Python 阅读次数：
本文字数： 2.7k 阅读时长 ≈ 2 分钟

使用 python 遍历文件目录，批量重命名文件，已支持功能包括

  • 添加前后缀，支持默认序列（输入 seq ）和自定义
  • 选择文件类型
  • 选择是否遍历子目录
  • 选择是否保留原文件名（python读取文件名顺序与文件管理器不同）
  • 如果已存在序列，可使用正则替换序列格式
